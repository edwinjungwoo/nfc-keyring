# 성경 NFC 키링 프로젝트 ✝️

**교회 권사님들을 위한 매일 바뀌는 오늘의 말씀 시스템**

성경 구절을 NFC 태그에 저장하고, 매일 랜덤으로 업데이트되는 말씀을 스마트폰으로 간편하게 읽을 수 있는 키링 프로젝트입니다.

---

## 🎯 프로젝트 특징

### ✝️ 오늘의 말씀
- 매일 자동으로 바뀌는 은혜로운 성경 구절
- 날짜 기반으로 매일 같은 말씀 제공
- 30+ 엄선된 추천 구절

### 📱 시니어 친화적 디자인
- **궁서체 폰트**: 성경책 느낌
- **큰 글씨**: 38px 본문, 편안한 가독성
- **높은 줄간격**: 2.2배 간격
- **큰 버튼**: 터치하기 쉬운 크기
- **명확한 색상**: 갈색 + 금색 조합

### 🏷️ 카테고리별 말씀
- 위로, 감사, 평안, 믿음, 소망, 사랑, 축복
- 각 카테고리별 엄선된 구절들

### 🔖 NFC 태그 지원
- 키링에 태그하면 특정 말씀으로 이동
- 개인별 맞춤 말씀 설정 가능

## 📦 프로젝트 구조

```
bible-nfc-keyring/
├── database/              # SQLite 데이터베이스
│   └── bible.db          # 성경 66권, 31,102구절 + 추천 구절
├── schema/               # 데이터베이스 스키마
│   ├── 01_books.sql      # 성경 66권 정보
│   ├── 02_verses.sql     # 성경 본문 (개역한글판)
│   ├── 03_nfc_tags.sql   # NFC 태그 매핑
│   └── 04_featured_verses.sql  # 추천 구절 (30+개)
├── webapp/               # 웹 애플리케이션
│   ├── app.py           # Flask 서버
│   ├── requirements.txt  # Python 패키지
│   ├── run.sh           # 실행 스크립트
│   ├── static/
│   │   └── style.css    # 시니어 친화적 스타일
│   └── templates/       # HTML 템플릿
│       ├── index.html   # 오늘의 말씀
│       ├── category.html # 카테고리별 말씀
│       ├── tag.html     # NFC 태그 페이지
│       ├── random.html  # 랜덤 말씀
│       └── error.html   # 에러 페이지
├── scripts/             # 유틸리티 스크립트
│   └── init_database.sh # DB 초기화
└── docs/                # 문서
    ├── README.md        # 데이터베이스 가이드
    ├── WEB_GUIDE.md     # 웹 앱 사용 가이드
    └── example_queries.sql  # 예제 쿼리
```

## 🚀 빠른 시작

### 1. 웹 애플리케이션 실행

```bash
cd bible-nfc-keyring/webapp
chmod +x run.sh
./run.sh
```

### 2. 브라우저에서 접속

```
http://localhost:5000
```

스마트폰에서 접속:
```
http://[컴퓨터IP]:5000
```

## 📊 데이터베이스

### 테이블 구조

1. **books** - 성경 66권
   - 구약 39권 + 신약 27권
   - 책 이름, 구분 (구약/신약)

2. **verses** - 성경 본문
   - 31,102개 구절
   - 개역한글판

3. **featured_verses** - 추천 구절
   - 30+ 엄선된 말씀
   - 7개 카테고리 분류

4. **nfc_tags** - NFC 태그 매핑
   - 태그 ID와 구절 연결

### 데이터베이스 초기화

```bash
cd scripts
./init_database.sh
```

## 💡 활용 방법

### 1. 교회 권사님들 선물용
```
- NFC 태그 스티커 구매
- 예쁜 키링에 부착
- 각 권사님별로 맞춤 말씀 설정
```

### 2. 매일 QT 도구
```
- 매일 아침 오늘의 말씀 확인
- 스마트폰 바탕화면에 바로가기 추가
- 묵상 후 자매님들과 공유
```

### 3. 전도 도구
```
- 복음 관련 말씀으로 키링 제작
- 새신자 선물용
- 교회 홍보용
```

## 📱 NFC 키링 제작 가이드

### 필요한 재료
1. NFC 태그 스티커 (NTAG213/215/216)
2. 아크릴 키링 또는 금속 키링
3. 십자가 참 장식
4. 스마트폰 (NFC Tools 앱)

### 제작 순서
1. NFC Tools 앱 설치
2. 태그에 URL 쓰기: `http://your-server.com/tag/NFC-001`
3. 키링에 부착
4. 테스트

상세 가이드: [docs/WEB_GUIDE.md](docs/WEB_GUIDE.md)
