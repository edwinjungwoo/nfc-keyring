# 성경 NFC 키링 - 웹 애플리케이션 가이드

## 🙏 프로젝트 소개

교회 권사님들을 위한 **매일 랜덤으로 업데이트되는 성경 말씀** NFC 키링 시스템입니다.

### 주요 기능
- ✝️ **오늘의 말씀**: 매일 자동으로 바뀌는 은혜로운 말씀
- 📱 **시니어 친화적 디자인**: 궁서체 + 큰 글씨 + 성경책 스타일
- 🏷️ **카테고리별 말씀**: 위로, 감사, 평안, 믿음, 소망, 사랑, 축복
- 🔖 **NFC 태그 지원**: 키링에 태그하면 특정 말씀으로 이동
- 📤 **말씀 공유**: 카카오톡, 문자로 쉽게 공유

---

## 🚀 빠른 시작

### 1. 서버 실행

```bash
cd webapp
chmod +x run.sh
./run.sh
```

### 2. 브라우저에서 접속

```
http://localhost:5000
```

스마트폰에서 접속하려면:
```
http://[컴퓨터IP주소]:5000
```

예: `http://192.168.0.10:5000`

## 📱 사용 방법

### 일반 사용자 (권사님들)

1. **오늘의 말씀 보기**
   - 메인 페이지에 접속하면 오늘의 말씀이 표시됩니다
   - 매일 자동으로 바뀝니다 (날짜 기준)

2. **카테고리별 말씀 보기**
   - 화면 하단의 카테고리 버튼 클릭
   - 위로, 감사, 평안, 믿음, 소망, 사랑, 축복

3. **말씀 공유하기**
   - "말씀 공유하기" 버튼 클릭
   - 카카오톡이나 문자로 자매님들과 나눌 수 있습니다

4. **NFC 키링 사용**
   - 스마트폰으로 NFC 키링을 태그
   - 자동으로 해당 말씀 페이지가 열립니다

## 🎨 디자인 특징

### 시니어 친화적 요소

1. **큰 글씨**: 38px 본문 크기
2. **궁서체 폰트**: 성경책 느낌
3. **높은 줄간격**: 2.2배 간격으로 읽기 편함
4. **큰 버튼**: 26px 글씨에 넉넉한 패딩
5. **명확한 색상**: 갈색(#8b5a2b) + 금색(#d4af37) 조합

### 성경책 느낌

- 📜 크림색 배경 (양피지 느낌)
- ✝ 십자가 장식
- 🖼️ 테두리와 그림자 효과
- 📖 성경 구절 박스 스타일

## 🔧 API 엔드포인트

개발자를 위한 API 엔드포인트:

```
GET /                      - 오늘의 말씀 페이지
GET /random                - 랜덤 말씀 페이지
GET /category/{category}   - 카테고리별 말씀
GET /tag/{tag_id}         - NFC 태그 페이지

GET /api/daily            - 오늘의 말씀 JSON
GET /api/random           - 랜덤 말씀 JSON
GET /api/tag/{tag_id}     - 태그 말씀 JSON
```

예시:
```bash
curl http://localhost:5000/api/daily
```

## 📋 NFC 키링 제작 가이드

### 필요한 재료

1. **NFC 태그 스티커** (NTAG213/215/216)
   - 온라인 구매: 쿠팡, 네이버 쇼핑
   - 개당 500원~1000원

2. **키링 재료**
   - 아크릴 키링
   - 십자가 모양 참
   - 체인 또는 스트랩

3. **스마트폰 앱**: NFC Tools (무료)

### 제작 순서

1. **NFC 태그에 URL 쓰기**
   ```
   http://your-server.com/tag/NFC-001
   ```

2. **키링에 부착**
   - NFC 스티커를 키링 뒷면에 붙임
   - 십자가 장식 추가

3. **테스트**
   - 스마트폰으로 태그
   - 웹페이지가 열리는지 확인

### 선물용 패키징

- 투명 케이스에 담기
- 말씀 카드와 함께 포장
- "하나님의 말씀을 언제나 함께" 메시지

## 🆘 문제 해결

### Q: 서버가 시작되지 않아요
```bash
# Python3가 설치되어 있는지 확인
python3 --version

# Flask 설치
pip3 install flask
```

### Q: 스마트폰에서 접속이 안돼요
- 컴퓨터와 스마트폰이 같은 와이파이에 연결되어 있는지 확인
- 방화벽 설정 확인
- IP 주소를 정확히 입력했는지 확인

### Q: NFC 태그가 인식되지 않아요
- 스마트폰의 NFC 기능이 켜져 있는지 확인
- 태그에 올바른 URL이 쓰여졌는지 확인
- 태그를 스마트폰 뒷면 중앙에 가까이 대기

### Q: 말씀이 매일 바뀌지 않아요
- 브라우저 캐시 삭제
- 페이지 새로고침 (Ctrl+F5)
- 서버 시간대 확인
